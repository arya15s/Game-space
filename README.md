# Game Space

A lightweight browser-based game portal built with HTML, CSS, and JavaScript.

## Live demo
https://arya15s.github.io/Game-space/

## Files
- `index.html`
- `styles.css`
- `script.js`

## How to run locally
```bash
git clone https://github.com/arya15s/Game-space.git
cd Game-space
python -m http.server 8000
```
